# Contributing

This repository’s contribution guidelines live at the project root: see **[`CONTRIBUTING.md`](../../CONTRIBUTING.md)** for the full process.

**Code review:** All changes merge via pull request. Maintainers review for correctness, style, and fit with the codebase; address feedback before merge.

**Testing:** Add or update tests for behavior you change. Run the test suite locally and ensure it passes before requesting review; CI must stay green.
